# MetaTrack

MetaTrack es una aplicación web moderna y minimalista diseñada para ayudarte a organizar tus objetivos, dividirlos en metas pequeñas y crear hábitos diarios que impulsen tu progreso.  
Su interfaz intuitiva te permite visualizar cómo avanzas, marcar hábitos completados y mantener un seguimiento claro de todo lo que quieres lograr.

Con MetaTrack puedes:
- Crear objetivos principales.
- Dividir cada objetivo en metas específicas.
- Registrar hábitos diarios para cada meta.
- Ver tu progreso general en tiempo real.
- Guardar automáticamente toda tu información con LocalStorage.

MetaTrack combina simplicidad, estética y funcionalidad para que avanzar hacia tus metas sea más fácil y motivador.
